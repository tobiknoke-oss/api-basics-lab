\## Issue 1 – Erste API



\*\*Endpoint:\*\* https://jsonplaceholder.typicode.com/todos  

\*\*Methode:\*\* GET  

\*\*Parameter:\*\* userId=1  

\*\*Statuscode:\*\* 200



\### Was bedeutet das?

\- Ich frage eine Liste von Todos ab und filtere nach userId=1.



\### JSON erklärt

\- `\[]` = Array (Liste)

\- `{}` = Objekt (ein Eintrag)

\- Beispiel-Felder:

&nbsp; - `userId` (Number): Nutzer-ID

&nbsp; - `title` (String): Titel

&nbsp; - `completed` (Boolean): true/false
